# UPI Fraud Detection

A fraud detection application for UPI transactions, combining a React/Vite client with a Node server and a Python machine learning service.

## Structure

- `client/` — frontend application built with React, TypeScript, and Tailwind UI components.
- `server/` — backend routes, static assets, and fraud detection utilities.
- `ml_service/` — Python model training and inference components.
- `shared/` — shared schema definitions.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development environment:
   ```bash
   npm run dev
   ```
